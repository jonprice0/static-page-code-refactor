# static-page-code-refactor


## Objective

> In an effort to learn best practices I refactored this less-than-desirable codebase. The html suffered from div-itis and the CSS was needlessly redundant. The primary goal was to improve the code as much as possible without effecting the appearance of the site at all. I added semantic elements, alt attributes for the images, and consolidated the CSS.


## A teachable moment...

> Among the questions that popped up during the refactor was this: what are the distinct use cases for `<section>` and `<article>`? I think I've sorted those out correctly... Have a look for yourself!
